# retmorse .-. . - -- --- .-. ... .

Tradutor de código morse.

## Como instalar

Instalar:

```bash
npm install retmorse
```

Usar:

```js
import { retmorse } from "retmorse";

console.log(retmorse.toMorse("lorem ipsum"));
// output: .-.. --- .-. . -- / .. .--. ... ..- --

console.log(retmorse.toTxt(".-.. --- .-. . -- / .. .--. ... ..- --"));
// output: lorem ipsum
```
